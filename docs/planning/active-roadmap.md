# Ferrum2D 활성 개발 로드맵

이 문서는 Ferrum2D에서 앞으로 수정하거나 신규 개발할 항목의 단일 기준 문서다. `feature-candidates.md`와 `refactor-roadmap.md`는 템플릿으로 유지하고, 실제 활성 작업은 이 문서에서 관리한다.

완료된 P0~P3 작업은 이 문서의 활성 목록에 남기지 않는다. 완료 기능의 사용법, public contract, 검증 기준은 `README.md`, `docs/engine`, `docs/development`, smoke 결과를 우선한다.
