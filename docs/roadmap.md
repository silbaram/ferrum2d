# Ferrum2D 로드맵 (초안)

## Phase 0: 저장소/문서 초기화
- 프로젝트 목적 및 MVP 범위 합의
- 아키텍처 문서 초안 확정
- 디렉터리 구조 생성

## Phase 1: 코어 루프 골격
- Rust core 기본 루프 인터페이스 정의
- WASM 바인딩 최소 인터페이스 설계
- TypeScript 초기 부트스트랩 구성

## Phase 2: 렌더링 최소 기능
- WebGL2 기반 2D 렌더링 파이프라인 최소 구현
- 스프라이트 또는 기본 도형 렌더링 검증

## Phase 3: MVP 예제(2D 탑다운 슈터)
- 입력/이동/발사/충돌 연결
- 게임 상태 표시(점수/시간)
- 기본 플레이 루프 성립

## Phase 4: 안정화
- 문서 정리 및 코드 리뷰 기준 적용
- MVP 범위 충족 여부 점검
